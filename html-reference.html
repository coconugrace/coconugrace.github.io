HTML

The HTML tags are used at the very beginning and end of your code. It signals the computer that your code is written in HTML.


Header


Title

A title is used as a title over paragraphs, images, etc. The title tags go around the content you want to add weight to. A default title is bolded and enlarged.


Body

The body is the place where the most of you content is placed- including headings, paragraphs, images, tables, etc.


Headings

Headers are used to signify importance over other content. The header tags go around the words you want to enlargen. Headers are bigger and bolder. There are six types of headers- h1 being the biggest header, and h6 being the smallest.


Paragraph

When you have a great amount of writing to include, you probably want to make it a paragraph. Paragraphs are surrounded by its paragraph tags and ae often styled.


Breaks

Breaks create spaces between lines. For example, if you want to write an acrostic poem, you don't want all the letters to be squished up on one line. Breaks make adding a new line possible.


Lines

This element is similar to a break, because is indicates a significant pause in the text. The only different is that a line runs across the page to separate the text.


Comments

People often use comments as personal note in their code. Comments cannot be seen on the page, but only in the code. Coders also comments to hash code out in order to debug their program.
