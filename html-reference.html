HTML

The HTML tags are used at the very beginning and end of your code. It signals the computer that your code is written in HTML.

Syntax

<html>All your html code belongs here</html>


Head

The head is located before the body, containing metadata (data about the page). Whatever is put in the head does not actually appear on the page. The head includes things like the title or style of the page.

Syntax

<head>Metadata about the page</head>


Title

The title is found in the head. It determines the title of the page (which the user can see on the tab at the top), as well as in the results of a search engine (such as Google) and in favorites/bookmarks.

Syntax

<title>Title of page</title>


Body

The body is the place where the most of you content is placed- including headings, paragraphs, images, tables, etc.

Syntax

<body>All the content in the page</body>


Headings

Headings are used to signify importance over other content. The heading tags go around the words you want to enlargen. Headings are bigger and bolder. There are six types of headings- h1 being the biggest,, and h6 being the smallest.

Syntax

<h4>This would be a size 4 heading</h4>


Paragraph

When you have a great amount of writing to include, you probably want to make it a paragraph. Paragraphs are surrounded by its paragraph tags and ae often styled.

Syntax

<p>This is your paragraph</p>


Breaks

Breaks create spaces between lines. For example, if you want to write an acrostic poem, you don't want all the letters to be squished up on one line. Breaks make adding a new line possible. Unlike paragraphs, breaks don't need an end tag.

Syntax

<br>


Lines

This element is similar to a break, because is indicates a significant pause in the text. The only different is that a line runs across the page to separate the text.

Syntax

<hr>


Comments

People often use comments as personal note in their code. Comments cannot be seen on the page, but only in the code. Coders also comments to hash code out in order to debug their program.

Syntax

<!-- This is your comment -->


Links

When coders want to redirect the user to a new place, they often use hyperlinks to do so. Hyperlinks can be placed on text or images, that when clicked, redirects to a new place on the web.

Syntax

<a href="url you want to go to">the text that appears</a>


Images

To insert images into your website, you must paste the image link into your code. You can resize the image by setting the height and width of it within the image tag. Images also usually have alternative text that appears when the image cannot be retrieved.

Syntax

<img src="image url" alt="alternative text">


Lists

Lists are used to add bullet points or number your text. There are many different types of lists, including different fonts of unordered lists, as well as lists numbered by lowercase Roman numerals, uppercase Roman numerals, and Arabic numbers. Lists can be as long as you want (of course don't try to crash your computer though).

Syntax

Ordered list:

<ol>
<li>content of bullet 1</li>
<li>content of bullet 2</li>
</li>content of bullet 3</li>

Unordered list:

<ul>
<li>content of bullet 1</li>
<li>content of bullet 2</li>
</li>content of bullet 3</li>

Inserting tables are generally complicated and require more work. Every table has its start and end tag, as well as tags indicating each new row and data cell.
